# Export Miraee Screens from Figma

## Quick Export (Recommended)

1. Open the Figma file: https://www.figma.com/design/Dlr2gRhKAeDQTBwJ9WnPA2/New-miraee
2. For each screen below, right-click the frame → **Export** → PNG @ 2x
3. Save to `public/images/projects/miraee/` with the exact filename
